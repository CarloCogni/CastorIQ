How Look-ahead Works

What is it?

Look-ahead shows a rolling window of schedule work — what starts, is in progress, and finishes across the next 2–6 weeks. Critical-path tasks are flagged with a CP badge. The 3D preview pane shows how linked model elements would appear at each week's snapshot.

Lifecycle

Import a schedule in Data Sources, link tasks to IFC elements in 4D Link, then open Look-ahead. Week cards are built from today's Monday forward. Expand a week to see task buckets and update the 3D preview colors for that week.

3D preview

  • Uses the latest processed IFC for this project.
  • Colors are a week-scoped 4D preview (complete, in progress, delayed, not started) — not full simulation playback.
  • Click a linked task row to highlight its model elements in amber.
  • Unlinked tasks show a note — linking is done in the 4D Link tab.

Who can do what

Any project member with schedule access can view Look-ahead and the 3D preview. Creating or editing task-to-model links requires modify access on the 4D Link tab.

Caveats

This is a read-only planning view — no links are created here. Preview coloring depends on schedule dates, bindings, and a processed IFC file. For timeline scrubbing, playback, and link review, use 4D Link.
