How IFC Insights works

What is it? IFC Insights runs 10 automated QA/QC checks on your IFC model to catch issues before they break 4D linking, exports, or coordination.

Checks run:

  • Critical Elements without storey, geometry outside storey, null/duplicate GlobalIds, duplicate products
  • Warning Missing ActivityCode, empty storeys, elements with no geometry, missing level
  • Info Missing material, empty property sets

How to use: The checks run automatically when you open this tab. Click Re-run checks after fixing issues in your authoring tool and re-uploading. Use Export CSV to share the issue report with your team.

Caveats: Checks run on the most recently processed IFC file for this project. Re-upload and re-process to get results for an updated model.
