How IFC Insights works

Readiness Rings — show what percentage of your IFC elements have Activity ID (4D) and Cost (5D) data populated. Green ≥ 80%, Amber 40–79%, Red < 40%.

Breakdown Cards — element counts grouped by Level, Element Type, and Material. Loaded live from the database.

Activity Schedule Summary — table of all unique Activity ID / Activity Name pairs found in element properties, with element counts. Use the search box to filter.

Cost Summary — appears only when Cost or Unit Cost properties exist. Shows total cost and breakdown by element type.

QA/QC Checks — 10 automated quality checks run on the IFC file. Click Re-run after fixing issues and re-uploading.

Engine Selector — Castor Engine uses the main analysis pipeline. Islam Engine is experimental.
