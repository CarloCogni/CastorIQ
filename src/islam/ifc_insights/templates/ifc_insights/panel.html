How IFC Insights works

Readiness Rings — show what percentage of your IFC elements have Activity ID (4D) and Cost (5D) data populated. Green ≥ 80%, Amber 40–79%, Red < 40%. The count beneath each ring shows how many elements are missing that property.

View Issues → — opens the IFC Issues tab with four detailed audit sections: Missing Activity ID, Missing Cost, Activity ID & Name Audit, and Levels Health.

Breakdown Cards — element counts grouped by Level, Element Type, and Material. Loaded live from the database.

IFC Health Checks — detailed quality checks (severity, global ID, type, issue, suggested fix) are now in the IFC Issues tab under Section 0.

Engine Selector — Castor Engine uses the main analysis pipeline. Islam Engine is experimental.
