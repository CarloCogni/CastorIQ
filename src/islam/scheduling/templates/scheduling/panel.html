How TimeLiner works

What is it? TimeLiner links your construction schedule to IFC model elements — exactly like Navisworks TimeLiner — enabling 4D simulation of your build sequence.

Workflow:

  1. Data Sources — upload a schedule file (.xlsx, .xer, or .xml). The system parses it and uses AI to validate it looks like a real construction programme.
  2. Attach — link tasks to IFC elements. Use Smart Pipeline (4-layer: exact → normalised → heuristic → embedding) or the legacy AI / parameter methods. Smart Pipeline auto-accepts high-confidence matches and flags lower-confidence ones for review.
  3. Review — inspect every binding the Smart Pipeline created. Accept or remove individual links, bulk-accept all bindings with ≥ 95% confidence, manually link tasks that were left unlinked, and export the full binding table as CSV.
  4. Gantt — review the bar chart. Green ✅ = all elements linked, amber ⚠️ = partial, red ❌ = unlinked. Click any bar to highlight that element in the 3D viewer.
  5. Simulate — drag the date slider to animate the build sequence in 4D in the 3D viewer.
  6. EVM — Earned Value Management dashboard. Shows SPI, CPI, EAC, and VAC KPI cards plus a cumulative S-curve (PV, EV, AC). Uses task costs when available; falls back to duration weighting when no costs are in the schedule.
  7. Look-ahead — Rolling window showing what starts, is in progress, and finishes across the next 2–6 weeks. Critical path tasks are flagged with a CP badge. Click a week card to expand/collapse it.

Caveats: Smart Pipeline matching is probabilistic — always review heuristic and embedding results in the Review tab before simulating. Elements must be processed (IFC Insights complete) for best coverage. EVM actual cost (AC) is estimated as EV × 1.05; replace with real cost data for accurate CPI.
