Import assets from CSV
CSV columns: global_id, name, ifc_type, spatial_global_id, location, asset_tag, manufacturer, model_number, serial_number, commissioning_date (YYYY-MM-DD), warranty_end (YYYY-MM-DD), classification_system, classification_code.
Each row is either linked (set global_id to match an IFC entity — existing assets are updated in-place) or orphan (leave global_id blank and set name for items not in the IFC model; ifc_type and the other columns are optional). See the help button next to "Asset Register" for the full column reference.
