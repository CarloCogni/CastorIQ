How the CastorIQ beta works
What is CastorIQ?

A bi-directional LLM assistant that bridges IFC building models and technical documentation. Ask cross-cuts the model + the docs and answers with citations. Modify proposes property changes in plain language and routes them through a risk-stratified review pipeline before touching the IFC.

How vetting works
  1. Apply. Fill in the form below — email, name, role, why you're interested.
  2. Review. A real human reads every application. Allow a few days.
  3. (Optional) Intro call. We may follow up for a 10-minute call to understand what you're trying to do.
  4. Account creation. If approved, we create your account and send a welcome email.
  5. Set password. The welcome email contains a one-time set-password link valid for 7 days.
What you get on approval
  • A sample IFC project pre-loaded so you can try Ask and Modify in seconds.
  • A daily token allowance for cloud LLM use (visible inside the app, resets at midnight UTC).
  • Direct access to the operator if anything is broken or you want the cap raised.
Caveats — read these
  • This is a private testing environment, not a polished product. Things will occasionally break.
  • When cloud LLM providers (Anthropic, Groq) are active, your prompts and the relevant context are sent to them. Don't upload anything you wouldn't share with those providers under their standard terms.
  • There is no public signup. The only path in is the application form on this page.
  • Geometry modifications are out of scope — CastorIQ handles properties, materials, classifications, psets. It does not move, add, or delete physical building elements.

CastorIQ is a private testing environment. Uploaded IFC models are processed by cloud LLM providers (Anthropic, Groq) when those providers are active. By applying you opt into a research / test setup.
